# Radius MVP


Минимальный рабочий RADIUS-сервер на Golang, на основе библиотеки layeh/radius.
Сервер принимает входящий RADIUS-запрос, разбирать пакет, выводить его содержимое в двух форматах:
   - JSON
   - XML
После обработки сервер всегда отвечает Access-Accept.

Библиотека атрибутов в подкаталоге ./dictionary

Логика сервера:
   - принимает RADIUS-запрос
   - вызывает функции преобразования пакета в JSON и XML строку
   - выводит полученные JSON и XML строки в консоль
   - возвращает клиенту ответ Access-Accept
